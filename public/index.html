Image Search Abstraction Layer

Usage
Latest Searches Endpoint: https://savagedev-image-search.herokuapp.com/latest

Response Format: 
{
    "term": "{most_recent_searched_term}",
    "when": "{date_when_searched}"
}
Image Search Endpoint: https://savagedev-image-search.herokuapp.com/{query}

This will respond with an array of 10 images.

{query} should be what you want to find images for.

Optional Parameters: ?offset={amount} Offset will allow you to paginate through the images.

Response Format: 
[
    {
        "url": "{link_to_image}",
        "snippet": "{short_description_of_image}",
        "thumbnail": "{link_to_image_thumbnail}",
        "context": "{link_to_page_where_image_was_found}"
    }
]
